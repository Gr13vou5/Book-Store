# BookStore

A web-based e-commerce application for selling books.

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Set up environment variables:
   Copy `.env.example` to `.env` and fill in the required values (like Mailtrap credentials if needed).

3. Start the development server:
   ```bash
   npm run dev
   ```

## Features
- Browse and search book catalog
- Shopping cart and checkout flow
- User authentication (Register/Login/Password Reset)
- Admin dashboard to manage products and locations
